## Getting Started

```bash
$ docker run --init -it --rm blacktop/webkit:snapshot

>>> print("HALP!");
HALP!
```

### Run a javascript file

```bash
$ cat test.js
print(1+1);
```

```bash
$ docker run --init -it --rm -v `pwd`:/data blacktop/webkit:snapshot /data/test.js
2
```
